Feature Documentation

Project: E-commerce Analytics Platform
Architecture: Medallion (Bronze → Silver → Gold)

1. Customer Feature
Source Tables

Bronze: customers

Silver: customers_clean

Gold: dim_customer

Description

Represents customer master data used for sales analysis, customer segmentation, and lifetime value calculation.

Key Columns
Column	Description
customer_id	Unique customer identifier
first_name	Customer first name
last_name	Customer last name
email	Contact email
country	Customer country
city	Customer city
effective_from	SCD Type 2 start date
effective_to	SCD Type 2 end date
is_current	Active record flag
Transformations

Removed duplicate customers

Standardized text fields (lowercase, trimmed)

Implemented SCD Type 2 to track historical changes

Business Usage

Customer Lifetime Value (CLV)

Repeat customer analysis

Customer behavior tracking
